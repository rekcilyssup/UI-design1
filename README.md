# UI-design1

A responsive web application built from a Figma design assignment.

## Features
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Modern Aesthetics**: Clean, white-themed layout with professional typography.
- **Interactive Elements**: Custom buttons, navigation, and feature cards.
- **Vercel Deployment**: Automatically deployed for easy viewing.

## Tech Stack
- HTML5
- Vanilla CSS
- Google Fonts (Roboto)

## Project Structure
- `index.html`: Main structure of the application.
- `styles.css`: Custom styling and design system.
- `assets/`: Image assets and illustrations.

## Deployment
The project is live at: [https://figma-assignment-livid-mu.vercel.app](https://figma-assignment-livid-mu.vercel.app)
